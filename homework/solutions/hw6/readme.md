# Budget Planner CLI

## Running the Budget Planner CLI

1. **Navigate to the project directory**:

```bash
cd /bootcamp/homework/solutions/hw4
```
2. **Run the application**:

```
python -m solution.main
```

3. **Running Unit Tests**

```
pytest 
```
