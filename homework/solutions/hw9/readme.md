# Budget Planner CLI

## Running the Budget Planner CLI

1. **Navigate to the project directory**:

```bash
cd /bootcamp/homework/solutions/hw8
```
2. **Run the application**:

```
Terminal 1 - Start the API:

RUN -> fastapi dev solution/api/main.py


Terminal 2 - Start the UI:

RUN -> python main.py
```


3. **Running Unit Tests**

```
pytest 
```

**Note: This is the initial version of the Budget Planner CLI.** 

**Development is still in progress, and features may be incomplete or updated.**
